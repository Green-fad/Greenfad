Types de Containers

Container Standard

Largeur maximale de 1200px avec padding responsive. Idéal pour la plupart des contenus.

.container { width: min(95%, 1200px); margin: 0 auto; padding-left: var(--fluid-spacing-md); padding-right: var(--fluid-spacing-md); }

Container Étroit

Largeur maximale de 800px. Parfait pour les articles et le contenu textuel.

.container-narrow { width: min(90%, 800px); margin: 0 auto; }

Container Large

Largeur maximale de 1400px. Idéal pour les dashboards et les interfaces complexes.

.container-wide { width: min(98%, 1400px); margin: 0 auto; }

Guide d'Utilisation

CSS Grid

Utilisez les classes .grid avec .grid-1, .grid-2, .grid-3, .grid-4 ou .grid-auto pour créer des layouts en grille.

Flexbox

Combinez .flex avec .flex-center, .flex-between, .flex-column, .flex-wrap pour des alignements flexibles.

Responsive

Utilisez .show-mobile, .show-tablet, .show-desktop et leurs équivalents .hide-* pour contrôler la visibilité.

Containers

Choisissez entre .container (standard), .container-narrow (étroit) et .container-wide (large) selon vos besoins.
